## Install

With Go version `>= 1.16`:

```sh
go install github.com/haunt98/resolution-go/cmd/resolution@latest
```

With Go version `< 1.16`:

```sh
GO111module=on go get github.com/haunt98/resolution-go/cmd/resolution
```

## Usage

```sh
resolution [width] [height] [diagonal]
```
